grid

The grid comes with three predefined columns: 33%, 50% and 75%. According to that, class-names are _33, _50 and _75, which need to be wrapped in a grid as seen below. Additionally the paddings between coloumns can be modified, using paddings from the spacing module. When used with paddings, the grid will automatically re-calculate to fit 100% of the width, minus any given padding.

headlines

Headlines reach from h1 through h5 and like all text elements are rendered with system fonts. They come with a predefined margin of 0.7em top and bottom. Headlines don't require any additional classes but can be extended with colors and other components easily.
