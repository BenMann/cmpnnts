cmpnnts spacing

v0.1.0

The spaceing component is very helpful in combination with all other components, as it provides predefined margins and paddings. You cna either choose certain sides of the element that a space should be applied to, or apply it to every side. Margins use the class-name mX and paddings can be set with pX, where X is a number from 0 to 7. The different spacings are as follows:

0 — .25rem — 0.8rem — 1rem — 1.8rem — 3rem — 6rem — 12rem

When used with the grids component, the grid will automatically account for the given padding like this: 


._25.p7 {
  max-width: calc(25% - 24rem);
}


npm install cmpnnts-spacing
