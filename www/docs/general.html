cmpnnts general

v0.1.0

The general component is a rather big collection of utility classes that can be useful in combination with most other components. The component contains default fonts on the html and body tag, classes like .db to set an element to display: block; and width as well as type modifiers. In the near future it is planned to split the general component into smaller ones, enabling developers to require even less CSS. 


html,body {
  font-family: -apple-system, BlinkMacSystemFont, sans-serif;
  color: rgba(0,0,0,0.75);
}

/* utilities */
.db {
  display: block;
}

.dilb {
  display: inline-block;
}

/* width modifiers */
.narrow,.nw {
  max-width: 20em;
}

.normal,.n {
  max-width: 30em;
}

.wide,.w {
  max-width: 35em;
}

/* type modifiers */
.min {
  font-size: 0.8em;
}

  font-size: 0.9em;
}

.plus {
  font-size: 1.5em;
}

.max {
  font-size: 1.6em;
}

.truncate {
