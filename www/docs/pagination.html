cmpnnts paginations

v0.1.0

The pagination module comes without much fluff and offers just a baisc list of numbers that correspond to your pages, no arrows etc. To use the pagination you need a container element with a class of .pgntn and elmements inside with a class of .pg for page. It is assumed the pages are clickable, thus the cursor is always set to pointer.

1 2 3 4 5 6 

.pgntn:before,
.pgntn:after {
    content: " ";
    display: table;
}
.pgntn:after {
    clear: both;
}
.pgntn {
    *zoom: 1;
}

.pgntn .pg {
  display: inline-block;
  float: left;
  padding: 0.5rem 1rem;
  border-top: 1px solid #000;
  border-bottom: 1px solid #000;
  cursor: pointer;

}

.pgntn .pg {
  border-right: 1px solid #000;
}

.pgntn .pg:nth-of-type(1) {
  border-left: 1px solid #000;
}


npm install cmpnnts-paginations
