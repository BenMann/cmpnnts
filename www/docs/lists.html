cmpnnts lists

v0.1.0

The list module sets smart defaults for ordered and unordered lists, replacing the default margin and padding and giving its elments a better indenting as well as vertical spacing.
You can use it by applying a class of lim1 or lim2 to your lists root. Additionaly you can set a list-style: none with the class .lsn

  • I'm an li
  • I'm an li
  • I'm an li 

ul,ol {
  margin: 0;
  padding: 0;
}

.lim1 li{
  margin-left: 1rem;
}

.lim2 li{
  margin-left: 2rem;
}

.lis li{
  margin: 0.8em 0;
}

.lis1 li{
  margin: 1em 0;
}

.lis2 li{
  margin: 2em 0;
}

.lsn {
  list-style: none;
}


npm install cmpnnts-lists
