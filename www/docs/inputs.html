cmpnnts inputs

v0.1.0

Th einputs component provides a minimal styling for inputs, which includes :active and :focus states. To use it, just put a .inpt class on the respective elements. This component can also easily be combined with the border component. 


<input class="inpt" type="text">


npm install cmpnnts-inputs
