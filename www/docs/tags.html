cmpnnts tags

v0.1.0

The tags component is a really small set of rules to create inline-block elements with a border. Being inline-block, they will adjust to the screen-size automatically, however you can also combine them with spacing to increase the margin between them. Also putting a hover on the tags seems like a good combination.
