The code component provides inline as well as block-level styles for code blocks and comes in three different colors. The default is grey but you can easily switch to a dark or light background with .dc or .lc. Code can also be combined with the class .dilb to inline it in text like this example...

$ git clone git://github.com/BenMann/cmpnnts
