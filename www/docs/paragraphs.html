cmpnnts paragraphs

v0.1.0

Paragraphs give you a predefined font-size and line-height, which is more readable than the default on basically every screen. It also let's you align text in every possible way and is best combined with the general component to allow defining the width of paragraphs.
This right here is a perfect example of a paragraph with width. 


.para,.p,p {
  line-height: 1.4;
  font-size: 1.1rem;
}

.tac {
  text-align: center;
}

.tal {
  text-align: left;
}

.tar {
  text-align: right;
}

.taj {
  text-align: justify;
}

.indent,.ip {
  text-indent: 1em;
}


npm install cmpnnts-paragraphs
