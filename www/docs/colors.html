cmpnnts colors

v0.1.0

The colors components gives you five predefines colors as well as the option to "color" elements transparent or gove them different shades of black or white. The predefined colors can be accessed with the following class-names, according to their names;

.cc, .color-charcoal
.cm, .color-mosque
.cr, .color-red
.cg, .color-green
.cl, .color-lightgreen
There are also classes for black and white (cw, cb). The general syntax convention goes like this:

Background: bg<color's first letter>
Color: c<color's first letter> 


.cc,.color-charcoal {
  color: #414141;
}

.bgc,.background-charcoal {
  background-color: #414141;
