cmpnnts hovers

v0.1.0

hovers offer the possibility to switch an element's opactiy with a predefined transition. The components also covers :active and :focus states, however more elaborate hover effects should be realized with custom CSS. To apply the hover effect to an element, just give it the class .hvr

Hover me! 

.hvr {
  opacity: 1;
  transition: opacity .15s ease-in;
}

.hvr:hover,
.hvr:focus {
  opacity: .5;
  transition: opacity .15s ease-in;
}

.hvr:active {
  opacity: .8; transition: opacity .15s ease-out;
}


npm install cmpnnts-hovers
