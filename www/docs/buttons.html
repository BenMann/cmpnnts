cmpnnts buttons

v0.1.0

The buttons component provides the bare minimum button in differnt sizes. It is probably a good idea combining it with the hover component to add some visual feedback to them. Buttons size can be modified via font-size, which is done with those classes: .min, .minus, .plus, .max
