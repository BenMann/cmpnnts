cmpnnts

v0.1.0

cmpnnts is a collection of tiny components written to consume as little bandwidth as possible. Every component can be used separately and is easy to use and beautiful to look at. Predefined classes make it easy to quickly design and build clean and responsive sites that load fast anywhere.

Classnames in cmpnnts are equally small as the actual styles they provide. If you're looking for descriptive names, modifiers or similar techniques to organize your CSS, cmpnnts might not be for you.

Philosophy

Because minimal classnames can be a blessing and a curse at the same time, good documentation is mendatory. This in­cludes stats about the size of each component and how to combine them.

Getting started

The easiest way to get all componnents is to install it via npm

$ npm install cmpnnts-all

However you can also just clone the repository and modify and rebuild components according to your needs.

$ git clone git://github.com/benmann/cmpnnts
